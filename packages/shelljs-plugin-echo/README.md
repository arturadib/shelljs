# shelljs-plugin-echo
## `echo` for ShellJS

## Installation

See [Using ShellJS Plugins](https://github.com/shelljs/shelljs/wiki/Using-ShellJS-Plugins).

## Usage

### echo(string [, string ...])

Examples:

```javascript
echo('hello world');
var str = echo('hello world');
```

Prints string to stdout, and returns a `ShellString`.
